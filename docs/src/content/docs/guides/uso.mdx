---
title: Cómo usar el componente
description: Guía para usar SimulationProvider y los componentes visuales del simulador.
---

## Instalación

```bash
npm install cpu-scheduler
```

## Uso básico

Importa `SimulationProvider` directamente desde el paquete y úsalo en cualquier página React o MDX:

```mdx
import { SimulationProvider } from 'cpu-scheduler';

<SimulationProvider
  algorithm="fcfs"
  processes={[
    { id: 'P1', arrival_time: 0, burst_time: 3 },
    { id: 'P2', arrival_time: 2, burst_time: 2 },
    { id: 'P3', arrival_time: 1, burst_time: 4 },
  ]}
  client:only="react"
/>
```

Sin `children`, el proveedor renderiza automáticamente: `ProcessTable`, `GanttChart`, `PlaybackControls` y `MetricsTable`.

## Componentes individuales

Si necesitas controlar la disposición, pasa los componentes como `children`:

```mdx
import { SimulationProvider, ProcessTable, GanttChart, PlaybackControls, MetricsTable } from 'cpu-scheduler';

<SimulationProvider algorithm="sjf" processes={[...]} client:only="react">
  <ProcessTable />
  <GanttChart />
  <PlaybackControls />
  <MetricsTable />
</SimulationProvider>
```

También puedes intercalar texto u otros elementos entre los componentes.
