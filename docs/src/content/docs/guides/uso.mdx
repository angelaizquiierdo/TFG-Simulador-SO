## Instalación

El módulo `cpu-scheduler` se consume como dependencia en tu proyecto React o Astro.

```bash
npm install cpu-scheduler
```

## Uso básico

Importa `SimulationProvider` y renderízalo con un algoritmo y una lista de procesos:

```tsx
import { SimulationProvider } from 'cpu-scheduler';

<SimulationProvider
  algorithm="fcfs"
  processes={[
    { id: 'P1', arrival_time: 0, burst_time: 3 },
    { id: 'P2', arrival_time: 2, burst_time: 2 },
    { id: 'P3', arrival_time: 1, burst_time: 4 },
  ]}
/>
```

Sin hijos, el proveedor renderiza los cuatro componentes visuales por defecto:
`ProcessTable`, `GanttChart`, `PlaybackControls` y `MetricsTable`.

## Composición personalizada

Si quieres intercalar contenido propio entre los componentes, pásalos como hijos:

```tsx
import {
  SimulationProvider,
  ProcessTable,
  GanttChart,
  PlaybackControls,
  MetricsTable,
} from 'cpu-scheduler';

<SimulationProvider algorithm="fcfs" processes={[...]}>
  <ProcessTable />
  <p>Texto explicativo entre componentes.</p>
  <GanttChart />
  <PlaybackControls />
  <MetricsTable />
</SimulationProvider>
```

Solo se renderizan los hijos que declares. El orden es libre.
