title: Configuración del componente
description: Referencia de las props de SimulationProvider y sus valores válidos.
---

`SimulationProvider` acepta las siguientes props:

## Props

### `algorithm` (obligatorio)

Nombre del algoritmo registrado. Debe coincidir con el valor de `IAlgorithm.name`.

```tsx
<SimulationProvider algorithm="fcfs" processes={[...]}>
```

Algoritmos disponibles:

| Nombre | Descripción |
|---|---|
| `"fcfs"` | First Come, First Served |
| `"sjf"` | Shortest Job First (no expropiativo) |
| `"ljf"` | Longest Job First (no expropiativo) |
| `"priority-np"` | Prioridad no expropiativa |
| `"srtf"` | Shortest Remaining Time First |
| `"priority-p"` | Prioridad expropiativa |
| `"round-robin"` | Round Robin |

### `processes` (obligatorio)

Array de objetos con la descripción de los procesos:

```ts
interface Process {
  id: string;           // identificador único, p.ej. "P1"
  arrival_time: number; // tiempo de llegada ≥ 0
  burst_time: number;   // duración de la ráfaga > 0
  priority?: number;    // prioridad (obligatorio para algoritmos de prioridad)
}
```

Si el array está vacío, el componente muestra un estado vacío sin error.

Si algún proceso tiene `burst_time = 0`, el componente muestra el error:
> "La ráfaga debe ser mayor que 0"

### `params` (opcional)

Parámetros adicionales del algoritmo:

```tsx
<SimulationProvider
  algorithm="round-robin"
  processes={[...]}
  params={{ quantum: 2 }}
>
```

| Parámetro | Tipo | Descripción |
|---|---|---|
| `quantum` | `number` | Quantum de tiempo para Round Robin (entero > 0) |

## Ejemplo completo

```tsx
import {
  SimulationProvider,
  ProcessTable,
  GanttChart,
  PlaybackControls,
  MetricsTable,
} from 'cpu-scheduler';

const processes = [
  { id: 'P1', arrival_time: 0, burst_time: 5 },
  { id: 'P2', arrival_time: 1, burst_time: 4 },
  { id: 'P3', arrival_time: 2, burst_time: 2 },
];

export default function RoundRobinDemo() {
  return (
    <SimulationProvider
      algorithm="round-robin"
      processes={processes}
      params={{ quantum: 2 }}
    >
      <ProcessTable />
      <GanttChart />
      <PlaybackControls />
      <MetricsTable />
    </SimulationProvider>
  );
}
```
