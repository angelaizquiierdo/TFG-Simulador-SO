# Cómo crear un algoritmo personalizado

El simulador acepta cualquier algoritmo que implemente la interfaz `IAlgorithm`.
No necesitas modificar el motor ni los algoritmos existentes.

## 1. Implementar `IAlgorithm`

```ts
import type { IAlgorithm, ReadyProcess } from 'cpu-scheduler';

export class MiAlgoritmo implements IAlgorithm {
  readonly name = 'mi-algoritmo';
  readonly preemptionMode = 'none' as const; // 'none' | 'on-better' | 'on-quantum'
  readonly requires = {};  // { priority?: boolean; quantum?: boolean }

  select(ready: readonly ReadyProcess[]): ReadyProcess {
    // Devuelve el proceso elegido para ejecutarse en CPU.
    // El array `ready` ya está ordenado por el desempate global del motor.
    const first = ready[0];
    if (first === undefined) throw new Error('Sin procesos en cola');
    return first;
  }
}
```

### Modos de expropiación

| `preemptionMode` | Comportamiento |
|-----------------|----------------|
| `'none'` | No expropiativo: `select()` solo se llama cuando la CPU queda libre |
| `'on-better'` | Expropiativo: `select()` se evalúa cada tick; si elige otro proceso, expropia |
| `'on-quantum'` | Round Robin: expropia al agotar el quantum |

### El campo `requires`

- `requires.priority = true` → el componente muestra el campo de prioridad en la UI.
- `requires.quantum = true` → el componente muestra el campo quantum (entero > 0).

## 2. Registrarlo

```ts
import { register } from 'cpu-scheduler';
import { MiAlgoritmo } from './mi-algoritmo';

register(new MiAlgoritmo());
```

## 3. Usarlo en el componente

```tsx
import { Simulator } from 'cpu-scheduler';

<Simulator
  algorithm="mi-algoritmo"
  processes={[
    { id: 'P1', arrival_time: 0, burst_time: 3 },
  ]}
/>
```

## Restricciones del algoritmo

- La función `select()` solo implementa la lógica de selección. No debe acumular historial,
  calcular métricas ni acceder al DOM.
- No usar `Math.random()` ni `Date.now()`: el simulador debe ser determinista.
- Solo puede importar desde `cpu-scheduler` (no desde `src/core/` internamente).
