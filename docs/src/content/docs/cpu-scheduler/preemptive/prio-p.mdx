---
title: Prioridad (Expropiativa)
description: Algoritmo de planificación por prioridad expropiativo. Menor número = mayor prioridad.
---

import { SimulationProvider } from 'cpu-scheduler';

## Descripción

Versión expropiativa del algoritmo de prioridad. En cada tick compara el proceso en CPU con los de la cola; si llega uno con mejor (menor) prioridad, expropia al actual.

## Escenario de ejemplo

<SimulationProvider
  algorithm="priority-p"
  processes={[
    { id: 'P1', arrival_time: 0, burst_time: 4, priority: 2 },
    { id: 'P2', arrival_time: 1, burst_time: 2, priority: 2 },
    { id: 'P3', arrival_time: 2, burst_time: 2, priority: 1 },
  ]}
  client:only="react"
/>
