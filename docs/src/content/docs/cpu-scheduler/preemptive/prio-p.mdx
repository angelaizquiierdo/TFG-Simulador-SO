## Descripción

**Prioridad expropiativa** selecciona el proceso con la **prioridad más baja** (número menor = mayor prioridad). A diferencia de la versión no expropiativa, si llega un proceso con mayor prioridad que el proceso en CPU, expropia la CPU inmediatamente.

- **Ventaja:** respuesta rápida a procesos urgentes.
- **Desventaja:** puede causar inanición de procesos con baja prioridad.

## Escenario de ejemplo

Procesos: P1 (llegada 0, ráfaga 4, prioridad 2), P2 (llegada 1, ráfaga 2, prioridad 2), P3 (llegada 2, ráfaga 2, prioridad 1).

Diagrama de Gantt esperado: P1[0–2], P3[2–4], P1[4–6], P2[6–8]
