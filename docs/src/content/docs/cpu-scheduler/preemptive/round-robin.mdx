---
title: Round Robin
description: Algoritmo expropiativo Round Robin con quantum configurable.
---

import { SimulationProvider, ProcessTable, GanttChart, PlaybackControls, MetricsTable, ProcessForm, AlgorithmParamsForm } from 'cpu-scheduler';

**Round Robin** asigna la CPU a cada proceso durante un tiempo máximo fijo llamado
**quantum**. Si el proceso no termina en ese tiempo, es expropiado y vuelve al final de la
cola de listos. Garantiza equidad y tiempo de respuesta acotado para todos los procesos.

<SimulationProvider
  algorithm="round-robin"
  processes={[
    { id: 'P1', arrival_time: 0, burst_time: 5 },
    { id: 'P2', arrival_time: 1, burst_time: 3 },
    { id: 'P3', arrival_time: 2, burst_time: 4 },
    { id: 'P4', arrival_time: 4, burst_time: 2 },
  ]}
  params={{ quantum: 2 }}
  client:only="react"
>
  <AlgorithmParamsForm />
  <ProcessTable />
  <GanttChart />
  <PlaybackControls />
  <MetricsTable />
  <ProcessForm />
</SimulationProvider>
