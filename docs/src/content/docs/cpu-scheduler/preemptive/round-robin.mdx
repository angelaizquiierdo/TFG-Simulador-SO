# Round Robin

**Round Robin** asigna a cada proceso un quantum de tiempo fijo. Al agotar el quantum,
el proceso vuelve al final de la cola y el siguiente recibe la CPU.

## Características

- **Modo**: expropiativo por quantum (`on-quantum`)
- **Criterio de selección**: FIFO
- **Campo requerido**: `quantum` (entero > 0)
- **Starvation**: no produce inanición

## Escenario de ejemplo

| Proceso | Llegada | Ráfaga |
|---------|---------|--------|
| P1      | 0       | 5      |
| P2      | 1       | 4      |
| P3      | 2       | 2      |

Quantum = 2. Diagrama esperado: P1[0–2], P2[2–4], P3[4–6], P1[6–8], P2[8–10], P1[10–11]
