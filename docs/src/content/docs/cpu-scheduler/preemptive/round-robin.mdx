## Descripcion

**Round Robin** asigna a cada proceso un quantum de tiempo fijo. Cuando el quantum
se agota, el proceso actual es expropiado y reingresa al final de la cola lista,
permitiendo que el siguiente proceso tome la CPU.

- Equitativo: todos los procesos reciben porciones iguales de CPU.
- El quantum determina el equilibrio entre tiempo de respuesta y sobrecarga de cambio de contexto.
- Si un proceso llega al mismo tick que el quantum expira, entra en cola antes que el proceso reencolado.

  ]}
  params={{ quantum: 2 }}
