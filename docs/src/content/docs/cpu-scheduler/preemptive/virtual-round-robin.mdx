---
title: Round Robin Virtual (VRR)
description: Algoritmo expropiativo Round Robin Virtual con E/S y cola auxiliar.
---

import { SimulationProvider, ProcessTable, GanttChart, PlaybackControls, MetricsTable, ProcessForm, AlgorithmParamsForm } from 'cpu-scheduler';

**Round Robin Virtual** (VRR) extiende Round Robin para tratar de forma justa a los
procesos intensivos en **E/S**. Cuando un proceso vuelve de E/S sin haber agotado su
quantum, entra en una **cola auxiliar** con el tiempo restante. La cola auxiliar tiene
**prioridad** sobre la cola principal, y el proceso recibe solo su sobrante de quantum
(no un quantum completo). Esto evita penalizar a los procesos que se bloquean por E/S.

VRR es el único algoritmo de este simulador que modela E/S: los procesos se bloquean,
hay dispositivo con cola FCFS y contención.
