---
title: Round Robin Virtual (VRR)
description: Algoritmo Round Robin Virtual con soporte de E/S.
---

import { SimulationApp } from 'cpu-scheduler';

El **Round Robin Virtual** (VRR) es una mejora sobre Round Robin que penaliza menos a los procesos con E/S. Cuando un proceso regresa de E/S, entra a una **cola auxiliar** que tiene prioridad sobre la cola principal. Además, su quantum en el siguiente dispatch equivale al tiempo de CPU que no pudo usar antes de la E/S (el _sobrante_), lo que le da una compensación justa.

**Características:**
- Expropiativo (`preemptionMode: 'io-return'`).
- Requiere `quantum` (`requires.quantum = true`) y E/S (`requires.io = true`).
- Usa `quantumFor()` para devolver el tiempo sobrante al regresar de E/S.
- Usa `onEvent()` para mantener la cola auxiliar y emitir mensajes ricos.
- Solo un dispositivo de E/S (FCFS). Si dos procesos llegan a E/S en el mismo tick, el segundo espera en cola.
