---
title: SRTF — Shortest Remaining Time First
description: Algoritmo expropiativo SRTF. Expropia si llega un proceso con ráfaga restante menor.
---

import { SimulationProvider, ProcessTable, GanttChart, PlaybackControls, MetricsTable, ProcessForm } from 'cpu-scheduler';

**SRTF** (Shortest Remaining Time First) es la versión expropiativa de SJF. En cada tick,
si llega un proceso con **ráfaga restante menor** que la del proceso en CPU, lo expropia
inmediatamente. Minimiza el tiempo de espera promedio teórico pero puede producir
inanición en procesos con ráfagas largas.

<SimulationProvider
  algorithm="srtf"
  processes={[
    { id: 'P1', arrival_time: 0, burst_time: 8 },
    { id: 'P2', arrival_time: 1, burst_time: 4 },
    { id: 'P3', arrival_time: 2, burst_time: 9 },
    { id: 'P4', arrival_time: 3, burst_time: 5 },
  ]}
  client:only="react"
>
  <ProcessTable />
  <GanttChart />
  <PlaybackControls />
  <MetricsTable />
  <ProcessForm />
</SimulationProvider>
