## Descripcion

**SRTF** es la version expropiativa de SJF. En cada tick, selecciona el proceso con
menor tiempo restante de rafaga. Si llega un proceso nuevo con rafaga mas corta que el
tiempo restante del proceso actual, este es expropiado.

- Minimiza el tiempo de espera promedio entre todos los algoritmos de planificacion.
- Puede causar inanicion de procesos con rafagas largas.
- Requiere conocer (o estimar) el tiempo restante de cada proceso.
