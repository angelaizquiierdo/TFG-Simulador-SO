import { SimulationProvider } from 'cpu-scheduler';

## Descripción

**SRTF** es la versión expropiativa de SJF. En cada tick reevalúa qué proceso tiene el menor tiempo restante; si llega un proceso mejor, expropia al actual.

## Escenario de ejemplo

<SimulationProvider
  algorithm="srtf"
  processes={[
    { id: 'P1', arrival_time: 0, burst_time: 8 },
    { id: 'P2', arrival_time: 1, burst_time: 4 },
    { id: 'P3', arrival_time: 2, burst_time: 2 },
    { id: 'P4', arrival_time: 4, burst_time: 1 },
  ]}
  client:only="react"
/>
