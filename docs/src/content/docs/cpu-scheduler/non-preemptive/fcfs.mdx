import { SimulationProvider } from 'cpu-scheduler';

## Descripcion

**FCFS** ejecuta los procesos en orden de llegada. Es no expropiativo: una vez que un
proceso entra en CPU, la ocupa hasta terminar su rafaga.

- Sencillo de implementar y entender.
- Puede causar el efecto convoy: procesos cortos esperan detras de procesos largos.
- No considera ni la duracion de la rafaga ni la prioridad.
