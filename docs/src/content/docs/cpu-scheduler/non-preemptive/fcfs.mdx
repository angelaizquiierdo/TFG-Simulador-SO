# FCFS — First Come, First Served

**First Come, First Served** es el algoritmo de planificación más sencillo.
Los procesos se ejecutan en el orden en que llegan: el primero en llegar es el primero en ejecutarse.

## Características

- **Modo**: no expropiativo
- **Criterio de selección**: orden de llegada (FIFO)
- **Starvation**: no produce inanición
- **Desventaja**: puede producir el "efecto convoy" si un proceso largo bloquea a otros cortos

## Escenario de ejemplo

| Proceso | Llegada | Ráfaga |
|---------|---------|--------|
| P1      | 0       | 3      |
| P2      | 2       | 2      |

Diagrama de Gantt esperado: P1[0–3], P2[3–5]
