title: SJF (Shortest Job First)

import { SimulationProvider } from 'cpu-scheduler';

## Descripción

**SJF** selecciona siempre el proceso con la ráfaga más corta cuando la CPU queda libre. No expropiativo.

## Escenario de ejemplo

<SimulationProvider
  algorithm="sjf"
  processes={[
    { id: 'P1', arrival_time: 0, burst_time: 5 },
    { id: 'P2', arrival_time: 1, burst_time: 2 },
    { id: 'P3', arrival_time: 2, burst_time: 4 },
    { id: 'P4', arrival_time: 3, burst_time: 1 },
  ]}
  client:only="react"
/>
