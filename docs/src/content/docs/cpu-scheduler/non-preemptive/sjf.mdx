# SJF — Shortest Job First

**Shortest Job First** selecciona el proceso con la ráfaga más corta cuando la CPU queda libre.

## Características

- **Modo**: no expropiativo
- **Criterio de selección**: menor `burst_time`
- **Starvation**: posible en procesos con ráfagas largas

## Escenario de ejemplo

| Proceso | Llegada | Ráfaga |
|---------|---------|--------|
| P1      | 0       | 5      |
| P2      | 1       | 2      |
| P3      | 2       | 4      |
| P4      | 3       | 1      |

Diagrama de Gantt esperado: P1[0–5], P4[5–6], P2[6–8], P3[8–12]
