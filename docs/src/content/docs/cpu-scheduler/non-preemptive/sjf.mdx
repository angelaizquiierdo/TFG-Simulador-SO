import { SimulationProvider } from 'cpu-scheduler';

## Descripcion

**SJF** selecciona el proceso con la rafaga mas corta entre los que estan en la cola lista
cuando la CPU queda libre. Es no expropiativo: una vez asignada la CPU, el proceso la
retiene hasta completar su rafaga.

- Minimiza el tiempo de espera promedio entre los algoritmos no expropiativos.
- Puede causar inanicion de procesos con rafagas largas si siempre llegan procesos cortos.
- Requiere conocer (o estimar) la duracion de la rafaga de antemano.
