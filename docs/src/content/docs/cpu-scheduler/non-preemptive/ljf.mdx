import { SimulationProvider } from 'cpu-scheduler';

## Descripcion

**LJF** selecciona el proceso con la rafaga mas larga entre los que estan en la cola lista
cuando la CPU queda libre. Es no expropiativo: una vez asignada la CPU, el proceso la
retiene hasta completar su rafaga.

- Maximiza la ocupacion de la CPU con trabajo de larga duracion.
- Puede causar tiempos de espera muy altos para procesos cortos.
- Es util para comparar el comportamiento contrario a SJF.
