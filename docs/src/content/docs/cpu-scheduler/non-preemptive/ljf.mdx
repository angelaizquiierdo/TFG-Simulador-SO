import { SimulationApp } from 'cpu-scheduler';

El **Longest Job First** (LJF) es el complemento de SJF: selecciona el proceso con la **ráfaga de CPU más larga** disponible en la cola de listos. No expropia.

**Características:**
- No expropiativo (`preemptionMode: 'none'`).
- Maximiza el uso de la CPU para trabajos largos, pero penaliza fuertemente a los procesos cortos (inanición en escenarios mixtos).
