title: LJF (Longest Job First)

import { SimulationProvider } from 'cpu-scheduler';

## Descripción

**LJF** selecciona siempre el proceso con la ráfaga más larga cuando la CPU queda libre. No expropiativo.

## Escenario de ejemplo

<SimulationProvider
  algorithm="ljf"
  processes={[
    { id: 'P1', arrival_time: 0, burst_time: 2 },
    { id: 'P2', arrival_time: 0, burst_time: 4 },
    { id: 'P3', arrival_time: 0, burst_time: 3 },
  ]}
  client:only="react"
/>
