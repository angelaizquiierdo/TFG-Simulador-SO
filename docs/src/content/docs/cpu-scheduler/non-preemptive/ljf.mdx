# LJF — Longest Job First

**Longest Job First** selecciona el proceso con la ráfaga más larga cuando la CPU queda libre.

## Características

- **Modo**: no expropiativo
- **Criterio de selección**: mayor `burst_time`
- **Nota**: maximiza el tiempo de retorno medio; útil en escenarios donde se priorizan las tareas largas

## Escenario de ejemplo

| Proceso | Llegada | Ráfaga |
|---------|---------|--------|
| P1      | 0       | 2      |
| P2      | 0       | 4      |
| P3      | 0       | 3      |

Diagrama de Gantt esperado: P2[0–4], P3[4–7], P1[7–9]
