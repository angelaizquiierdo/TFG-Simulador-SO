import { SimulationProvider } from 'cpu-scheduler';

## Descripción

**LJF** selecciona el proceso con la ráfaga más larga entre los que están en la cola
de listos. Es no expropiativo: una vez que un proceso entra en CPU, la ocupa hasta
terminar su ráfaga.

## Escenario de ejemplo

<SimulationProvider
  algorithm="ljf"
  processes={[
    { id: 'P1', arrival_time: 0, burst_time: 2 },
    { id: 'P2', arrival_time: 0, burst_time: 4 },
    { id: 'P3', arrival_time: 0, burst_time: 3 },
  ]}
  client:only="react"
/>
