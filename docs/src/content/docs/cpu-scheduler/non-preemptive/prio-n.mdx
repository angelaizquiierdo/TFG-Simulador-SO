## Descripción

**Prioridad no expropiativa** selecciona el proceso con la **prioridad más baja** (número menor = mayor prioridad) de entre los procesos listos. Es **no expropiativo**.

- **Ventaja:** permite controlar la urgencia de los procesos.
- **Desventaja:** puede causar inanición de procesos con baja prioridad (número alto).

## Escenario de ejemplo

Procesos: P1 (llegada 0, ráfaga 4, prioridad 4), P2 (llegada 0, ráfaga 3, prioridad 2), P3 (llegada 0, ráfaga 2, prioridad 1).

Diagrama de Gantt esperado: P3[0–2], P2[2–5], P1[5–9]
