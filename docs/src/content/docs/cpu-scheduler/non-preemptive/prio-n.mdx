---
title: Prioridad (No expropiativo)
description: Algoritmo de planificación por prioridad no expropiativo.
---

import { SimulationApp } from 'cpu-scheduler';

El algoritmo de **Prioridad No Expropiativo** selecciona el proceso con la **prioridad más alta** (número más bajo) en la cola de listos. Una vez en CPU, el proceso no puede ser interrumpido.

**Características:**
- No expropiativo (`triggers: {}`).
- Requiere el campo `priority` en cada proceso (`requires.priority = true`).
- Puede causar inanición de procesos con prioridad baja.

<SimulationApp
  algorithm="priority-np"
  processes={[
    { id: 'P1', arrival_time: 0, burst_time: 4, priority: 2 },
    { id: 'P2', arrival_time: 1, burst_time: 3, priority: 1 },
    { id: 'P3', arrival_time: 2, burst_time: 2, priority: 3 },
    { id: 'P4', arrival_time: 3, burst_time: 5, priority: 1 },
  ]}
  params = {{ priority: true}}
  client:only="react"
/>
