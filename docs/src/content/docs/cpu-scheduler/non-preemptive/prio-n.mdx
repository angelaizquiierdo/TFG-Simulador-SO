# Prioridad no expropiativa

Selecciona el proceso con **mayor prioridad** (número de prioridad más bajo) cuando la CPU queda libre.

## Características

- **Modo**: no expropiativo
- **Criterio de selección**: menor `priority` (número)
- **Campo requerido**: `priority` en cada proceso
- **Starvation**: posible en procesos de baja prioridad

## Escenario de ejemplo

| Proceso | Llegada | Ráfaga | Prioridad |
|---------|---------|--------|-----------|
| P1      | 0       | 3      | 3         |
| P2      | 2       | 2      | 2         |
| P3      | 1       | 4      | 3         |
| P4      | 2       | 2      | 2         |

Diagrama de Gantt esperado: P1[0–3], P2[3–5], P3[5–9]
