---
title: Prioridad (No Expropiativo)
description: Algoritmo no expropiativo por prioridad. Selecciona el proceso con mayor prioridad (menor número).
---

import { SimulationProvider, ProcessTable, GanttChart, PlaybackControls, MetricsTable, ProcessForm } from 'cpu-scheduler';

**Prioridad (NP)** selecciona el proceso con **mayor prioridad** (número de prioridad más
bajo). Es no expropiativo: una vez que el proceso seleccionado toma la CPU, la mantiene
hasta completar su ráfaga. En caso de empate en prioridad, se aplica desempate por
`arrival_time` e `id`.

<SimulationProvider
  algorithm="priority-np"
  processes={[
    { id: 'P1', arrival_time: 0, burst_time: 4, priority: 2 },
    { id: 'P2', arrival_time: 1, burst_time: 3, priority: 1 },
    { id: 'P3', arrival_time: 2, burst_time: 2, priority: 3 },
    { id: 'P4', arrival_time: 3, burst_time: 5, priority: 1 },
  ]}
  client:only="react"
>
  <ProcessTable />
  <GanttChart />
  <PlaybackControls />
  <MetricsTable />
  <ProcessForm />
</SimulationProvider>
